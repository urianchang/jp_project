Kanji Decoder

This application is design to help people find kanji that they see all over in the real world. It can be a real hassle to do this using the conventional method of drawing it out and searching around for it. There is a shirtcut however. It is called a SKIP code. This code categorizes the kanji and makes searching for it easy and painless when compared to the regular methods of finding kanji. And we will teach you how to use it!
